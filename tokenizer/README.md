## Java Tokenizer

Run the script `tokenizer.sh` inside script/java.

* for tokenizing java class file.

`bash tokenizer.sh java tokenizer/java/sample.java`

* for tokenizing java method.

`bash tokenizer.sh method tokenizer/java/sample.java`
